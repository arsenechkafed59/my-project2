ТЕСТ РУФЬЕ
Тест Руфье - это функциональная проба для оценки работоспособности сердца и тренированности организма при физической нагрузке. Тест прост в исполнении и не требует специального оборудования, поэтому подходит для массовых обследований.
Проект посвещён Тесту Фурье,где можно оценить работоспособность сердца,только в приложении.
